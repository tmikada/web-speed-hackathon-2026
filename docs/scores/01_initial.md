## 分析・課題

### FCP / LCP / SI がすべて 0 点
- **最重要課題**: バンドルが大きすぎてレンダリングが始まらない
- FCP・LCP・SI の 3指標で 0点 = 45点/ページが丸ごと失われている
- webpack バンドル最適化（コード分割・Tree Shaking・圧縮）が最優先

### TBT が低い or 0 点のページが多い
- JS実行ブロッキングは一部ページでスコアあり（利用規約: 30点満点）
- ホーム・動画・音声ページは TBT=0 → レンダリング自体が遅すぎる可能性

### CLS はほぼ安定
- 多くのページで CLS 満点または近い → レイアウト崩れは軽微

### DM ページ計測不可
- 認証フローに問題あり → サインインモーダルが動作していない可能性
